# vim配置
1、mkdir .vim  

2、`git clone https://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/Vundle.vim`  

3、download .vimrc from github  

4、vim  
     :PluginInstall  

### 效果  
鼠标可用  
滚轮可用  
F3 开启关闭树状菜单

mac下：
copy：1.visual模式下选中后 y
	  2.按alt同时选中要copy的区域
paste：command + v
- F5 快速运行  
- 函数自动补全  
- ctrl+c ctrl+v 可用：ctrl+c在视图模式下使用，ctrl+v在普通模式下使用  
- ctrl+a：全选
